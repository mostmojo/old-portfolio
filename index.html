Dice Roll

Dice rolling game built in vanilla JS. I used variable declarations, logical operators, conditions, for loops and DOM manipulation to change classes, images and modularise functional logic to display things to the UI. Went a step further with event handlers to add keyboard functionality for two player action. Whether you're a mouse ninja or a spacebar & enter key wizard, the loser has to face the consequences. YouTube clone

A ReactJS-based clone of YouTube

Youtube clone

Another fun project where I used ReactJS to build a YouTube interface clone. Used component driven development. Handled state in the App.js file and passed down props to a VideoList and VideoItem component. Additionally, used the YT API to fetch and manipulate data. As focus was React, I used semantic-ui CSS library to quickly style my components.

My asset pipeline architecture was done in SASS (BEM syntax), npm compilers and build scripts using node for all three projects. I also use zsh shell and home brew to make the terminal slightly more colorful. Oh, and the big devserver watch:sass command helps detect my errors along the way.
